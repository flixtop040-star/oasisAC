# 🌴 oasisAC — Next-Gen Cloud AI Anti-Cheat

> **Инновационный античит нового поколения для защиты игровых серверов.** > oasisAC использует мощь удаленных нейросетей для анализа поведения игроков в реальном времени. Никаких жестких правил — только умный поведенческий анализ без нагрузки на ваш сервер.

---

## 🚀 Почему именно oasisAC?

Большинство античитов можно декомпилировать, изучить их код и написать обход. С **oasisAC** это невозможно, потому что вся логика проверок находится на нашем защищенном ML-сервере.

* **🧠 Поведенческий ИИ (Behavioral AI):** Нейросеть анализирует микровибрации мыши, тайминги кликов и векторы движения. Она отличает человека от машины, выявляя даже самые скрытые и приватные читы.
* **☁️ Облачная архитектура:** Плагин на вашем сервере — это лишь легковесный агент, который собирает метрики. Вся тяжелая аналитика происходит на наших мощностях.
* **⚡ Нулевая нагрузка (0 TPS Impact):** Данные отправляются на сервер асинхронными батчами (пакетами). Античит абсолютно не нагружает основной поток игрового сервера.

---

## 🛠️ Основные функции

* **Умная система VL (Violation Levels):** Исключает ложные баны. Игрок блокируется только тогда, когда нейросеть уверена в нарушении на 99%.
* **Anti-Lag Fallback:** Античит автоматически снижает чувствительность или переходит на локальные базовые проверки, если у игрока высокий пинг или на сервере падает TPS.

## 📋 Команды и Права (Permissions)

| Команда | Описание | Право (Permission) |
| :--- | :--- | :--- |
| `/oasis help` | Показать меню помощи и статус соединения с облаком | `oasis.admin` |
| `/oasis info <игрок>` | Узнать текущий AI Score (индекс подозрительности) игрока | `oasis.moderator` |
| `/oasis alerts` | Переключить уведомления античита в чате | `oasis.notify` |
| `/oasis datacollect start legit <игрок>` | Пометить ложное срабатывание (отправляет лог на дообучение ИИ) | `oasis.moderator` |
| `/oasis reload` | Перезагрузить конфигурацию плагина | `oasis.admin` |

## 📦 Установка и Настройка

1. Скачайте последнюю версию `oasisAC.jar` из раздела **Releases**.
2. Поместите файл в папку `/plugins` вашего сервера.
3. Запустите сервер, чтобы сгенерировать конфигурационные файлы.
4. Откройте файл `plugins/oasisAC/config.yml` и вставьте ваш персональный API-ключ:
   ```yaml
   cloud:
     endpoint: "api site"
     api-key: "ВАШ_СЕКРЕТНЫЙ_КЛЮЧ_ЗДЕСЬ"
